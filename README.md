# get-twitter-media

## Usage

### Write keys in `main.js`

```javascript
const twitter = new Twitter({
	consumer_key: '',
	consumer_secret: '',
	access_token_key: '',
	access_token_secret: ''
});
```

### Run

e.g.) `node main.js iRis_s_yu`

`iRis_s_yu` is screen name of twitter that user favorites.
